# Remaining Mock Sources from CDISC Pilot

This package contains:
1. metadata/ - study, site, visit, treatment arm, expected assessment reference files
2. iwrs/ - randomization, treatment assignment, and drug dispensing files
3. central_lab/ - sample collection and lab result vendor files

These are designed to complement:
- edc_mock_from_cdisc_pilot.zip
- safety_mock_from_cdisc_pilot.zip

They intentionally include a few DQ/reconciliation scenarios for MVP testing.
