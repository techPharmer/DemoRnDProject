# Business Context

## Client Overview
A mid-sized biotech organization conducting multi-site clinical trials.

## Current Problem
Clinical trial data is received from multiple CROs/vendors in inconsistent formats.

Current challenges:
- inconsistent file schemas
- duplicate patient records
- missing values
- invalid dates
- inconsistent coding
- no central curated dataset
- manual reconciliation effort

## Business Impact
- delayed reporting
- unreliable analytics
- poor auditability
- manual engineering effort
